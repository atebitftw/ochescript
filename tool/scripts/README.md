## How to run scripts
### Use the batch file (while in the tool/scripts directory)
run {script_file}

### Use oche command if it's activated in your environment (see doc/getting_started.md
```bash
oche {script_file}
```

### If the bin/oche.dart is not executable, you can also:
```bash
dart run bin/oche.dart {script_file}
```
