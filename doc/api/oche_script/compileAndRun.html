Compiles and runs the given OcheScript source code.

Returns a Map<String, Object> containing the state exported by the script out() function, if any were used.

Example:

final result = await compileAndRun(r"print("Hello World!");");
  • preprocesser is the platform-specific IncludesPreprocesser to use for preprocessing the source code for any requested libraries.
  • outCallback is the callback to use for handling out() function calls at runtime.
  • dartFunctionCallback is the callback to use for handling dart() function calls at runtime.
  • initialGlobalState is the initial global state to use for the runtime. This essentially injects the states as global variables in the script.

For a full-featured example, see example/main.dart in the project source code.
